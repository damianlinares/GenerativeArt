//-- CÓMO MANIFESTAR LAS OBRAS (EJECUCIÓN) --//

Estas obras son portales vivos y necesitan ser servidas desde un pequeño espacio sagrado (un servidor local) para funcionar correctamente. No puedes simplemente abrir los archivos `index.html` en tu navegador.

Opción 1: Usando Visual Studio Code
1. Instala la extensión "Live Server".
2. Haz clic derecho sobre cualquier archivo `index.html` de una obra.
3. Selecciona "Open with Live Server".

Opción 2: Usando Python
1. Abre una terminal o línea de comandos en la carpeta raíz de este proyecto.
2. Ejecuta el comando: `python -m http.server` (para Python 3) o `python -m SimpleHTTPServer` (para Python 2).
3. Abre tu navegador y ve a la dirección: `http://localhost:8000`.
